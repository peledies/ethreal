# EthReal

### Install
```
npm install ethreal -g
```

### Useage
```
ethreal
```

### Output
```

║ Bitcoin  │ $5423.37 │ -3.56 │  12.27 ║
╟──────────┼──────────┼───────┼────────╢
║ Ethereum │ $305.438 │ -4.82 │  1.03  ║
╚══════════╧══════════╧═══════╧════════╝
```

### Supported Cryptocurriencies

EthReal currently displays the current market value in USD for the folowing:

- Bitcoin (BTC)
- Ethereum (ETH)
